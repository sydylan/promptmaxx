# promptmaxx

## Categories of Prompts

1. Ideate Prompts
Used for brainstorming, generating ideas, and determining usecases.

2. Prototype Prompts
Detailed step by step instructions on layout of your features and functionalities. Often paired with visual UI screenshots and elements.

3. Debug Prompts
Providing contextual files and specific areas of the code to look at. The description of the error or undesired behavior and debug methods you want the ai to use to tackle it.
